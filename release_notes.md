# Longboi Launcher: Niagara Overhaul (v1.1.0)

A major transformation bringing the premium Niagara style to Longboi Launcher.

### ✨ Key Features
- **Niagara-style UI**: Unified Home and All Apps experience with a clean, left-aligned vertical scrolling list.
- **Alphabet Scrubber**: Interactive persistent vertical scrubber for lightning-fast app navigation.
- **Smart Gestures**: Pull down from anywhere at the top of the list to expand the system notification shade.
- **Icon Customization**: Full support for third-party icon packs and several built-in sets (Glass, Minimalist).
- **Beautiful Settings**: Completely redesigned, intuitive settings app with a visual theme switcher and real-time previews.

### 🛠️ Improvements & Fixes
- **Stability**: Fixed critical `VIBRATE` permission crashes and Compose recomposition loops.
- **Visuals**: Enabled true wallpaper visibility via theme transparency and Material 3 optimizations.
- **Performance**: Streamlined the build system and modernized the internal app-catalog repository.
- **Automation**: Added a `make visual-audit` command for generating on-demand app screenshots.

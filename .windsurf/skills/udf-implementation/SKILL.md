# UDF Implementation Skill

Use this when creating ViewModels for Longboi Launcher features.

## Steps

1. Define the UI State data class (see `state-template.kt`)
2. Define the Event sealed interface (see `event-template.kt`)
3. Create the ViewModel with StateFlow and event handling (see `viewmodel-template.kt`)
4. Wire up the stateless Composable screen

## Resources

- `state-template.kt` - Template for UI state
- `event-template.kt` - Template for events
- `viewmodel-template.kt` - Complete ViewModel template
